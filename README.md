# Fruit Catcher Game 

This is a very simple and common game. You have a picnic basket and you catch the fruit falling from the sky and keep progressing through the levels. While the fruit is dropping, there are power ups and hazads to add some challenge to the game. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install VS Code to run the program.
2. Ensure you have Python 3.9 or newer installed on your machine.
3. Open your terminal or command prompt and install the required Arcade library by running: pip install arcade
4. Download or clone this repository to your machine.
5. Navigate to the folder containing the game and run the script using.

Instructions for using the software:

1. Main Menu: Press [SPACE] to start, [S] for Settings, or [ESC] to exit.
2. Gameplay Controls: Use the LEFT and RIGHT arrow keys to move your basket. Catch fruits to score points and advance levels.
3. Settings Menu: Press [F] to toggle Sound Effects, [M] to toggle Music, [C] to change your Basket Color, and the Arrow Keys to adjust movement sensitivity.
4. Game Over: Type your name using your keyboard and press [ENTER] to save your stats to the Top 5 Local Leaderboard!

## Development Environment
